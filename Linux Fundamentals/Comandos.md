
| Comando         | Uso                                                                                                                                   |
| --------------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| *sudo ls /root* | Hace una lista del contenido del directorio del superusuario (root)                                                                   |
| _--help_        | Te ayuda a ver cómo usar cada comando                                                                                                 |
| _mkdir_         | sirve para crear directorios nuevos                                                                                                   |
| *whoami*        | Te da tu nombre de usuario                                                                                                            |
| _ls_            | Te da una lista de los archivos                                                                                                       |
| _ip a_          | Te da tu dirección IP                                                                                                                 |
| _cat_           | muestra celdas de texto                                                                                                               |
| _passwd_        | cambia tu contraseña actual                                                                                                           |
| _touch nombre_  | Crea un archivo vacío y con el nombre que se le pone después                                                                          |
| _pwd _          | muestra los directorios que en los que se está trabajando                                                                             |
| _cd_            | Cd solo te va a enviar a tu directorio principal. Con la dirección de dónde te quieres dirigir te dirigirá al directorio seleccionado |
| _history_       | Verifica todo lo que acabas de hacer                                                                                                  |
| _man command_   | Te da toda la información del comando que estás utilizando                                                                            |
| _mandb_         | Te muestra la información que hay en man                                                                                              |
| _apropos_       | Te permite buscar comandos que pertenecen a una tarea específica basado en la descripción                                             |
| _pinfo_         | Es información adicional de los comandos                                                                                              |
| _tldr_          | es un comando nuevo que muestra ejemplos de líneas de comando                                                                         |
