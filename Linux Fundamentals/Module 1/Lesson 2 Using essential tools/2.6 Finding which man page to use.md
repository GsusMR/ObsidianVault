- man pages are summarized in the **mandb**
- Use **man -k** (or **apropos**) to search the **mandb**
- Type **sudo mandb** if you get "nothing appropriate"
- man mandb will guide you to another page, if the page has a 1 or an 8 the you are in the right direction
- with **sudo mandb** you can update the mandb in your system
- with **man -k user | less** you can have a description for the commands with less text, using space you can see each page of information
- with **man --k user | grep 8** we can find information about creating new users (for example). This lets you change the section of the man command
