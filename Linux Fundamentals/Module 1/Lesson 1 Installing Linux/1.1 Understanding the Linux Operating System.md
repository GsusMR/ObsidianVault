- It originate from UNIX, the leading operating system in the 1970's - 1990's 
- Fits in the movement started by Free Software Foundation (FSF). 
- In the 70's Unix became very comercial and expensive as a counterpart FSF started with the goal to provide software for free
- GNU, wihch means GNU Not Unix which mission is to provide free tools for Unix 
- In the 1980's, many free tools were already launched
- In 1992 Linus Torvalds added the Linux kernel to GNU 
- By combining the Linux kernel with other GNU tools, it was possible to install Linux as an actual operating system 
- To make an installation possible, there are multiple provided distributions
- Yggdrasil and Slackware were among the earliest Linux distributions
