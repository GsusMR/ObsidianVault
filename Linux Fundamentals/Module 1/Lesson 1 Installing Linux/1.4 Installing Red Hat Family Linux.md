**Requirements to install CentOs Stream**
- Download the latest version 
- Install in a virtual Machine or on Hardware
- Minimal RAM 1GB 
- Minimal Disk: 10GB
	- A second disk will be needed later in class
- Install using the "Server with GUI" pattern

1. Install CentOs Stream 9
2. Select an installation language 
3. Click on user settings and change root account
4. Create a Password
5. Click on Allow root SSH login with password
6. Click on done
7. Select Software Selection
8. Select server with GUI
9. Click on done
10. Click on selection destination
11. Click on done
12. Create an user 
13. Click on make it administrator 
14. Save the users credentials
15. Click on initialize installation
16. Click on reboot system
