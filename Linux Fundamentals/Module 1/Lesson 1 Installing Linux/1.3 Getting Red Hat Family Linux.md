 How to get any of the distributions from the red hat family
**If you want to work with red hat**
1. Go to [developers.redhat.com](https://developers.redhat.com/)
2. Log In or create an account
3. After login in, go to Products > Red Hat Enterprise Linux > Download RHEL at no-cost

**If you want to use CentOS**
1. Go to [centos.org](https://centos.org/)
2. Go to Download > CentOs Stream > Select the latest version and select the version x86_64

In the course its going to be used CentOS Stream because is the only distribution that reflects Red Hat Enterprise Linux 9 

**If you want to use Rocky Linux**
 1. Go to [rockylinux.org](https://rockylinux.org/es-ES)
 2. Click on download on the latest version x86_64
