- Download the latest version of Ubuntu from Ubuntu.com
- Install in a virtual Machine or on hardware
- Minimal RAM: 1 GB
- Minimal disk: 10 GB

1. Accept default
2. Click on install Ubuntu 
3. Select the language 
4. Click on the minimal installation
5. Install third party software 
6. Erase disk and install Ubuntu 
7. Select where you are
8. Create your user credentials
9. Click enter to restart
