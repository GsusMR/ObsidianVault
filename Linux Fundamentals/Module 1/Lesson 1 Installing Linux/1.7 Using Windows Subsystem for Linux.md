- Provides an Ubuntu instance that can be started as part of the Windows operating system 
- Is a Linux shell and Linux utilities that are on top of the windows kernel
- WSL uses part of the Windows operating system, and for that reason is different from other distributions regarding the boot procedure and kernel use 

 
 **To use it you need these requirements**
 - Enable HyperV in Windows Features 
 - Restart
 - Run powershell with Administrator privilegies
 - Use wsl --install
 - Restart
