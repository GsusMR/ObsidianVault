- **cut** is an easy-to-use command that filters out columns form text files. To do its work properly, you need to specify the field delimiter, using **-d**, this specify how cut can recognize fields
	- **cut -f 3 /etc/passwd** 
	- **cut -d : -f3 /etc/passwd** this is a delimiter to print what we want, in this case it was printed the third field on the search
- **sort** is often used with cut to sort the result of the cut command
- Use **sort -n** to sort in numeric order
	- **cut -d : -f3 /etc/passwd | sort**
	-  **cut -d : -f3 /etc/passwd | sort -n**
