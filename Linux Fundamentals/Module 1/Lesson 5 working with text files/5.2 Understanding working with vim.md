- After starting **vim**, it opens in command mode. Command mode is a mode in which you can use commands only, so if you start typing your text immediately it's not going to work until you hit an 'i' or 'o' 
- You can click on **Esc** to get back to command mode
	-  **:wq!** write, quit and don't complain. Makes sure that your changes are saved
	- **u** undo the last change
	- **Ctrl + r** re does the last undone change
	- **dd** deletes the current line 
	- **:%s/old/newg** replaces all occurrence of old with new
	- **:q!** quit without saving
	- **v** enter visual mode
	- **d** delete current selection
	- **y** copy the current selection
	- **3dd** delete 3 lines
	- **/text** search for text
	- **gg** move cursor to top of document 
	- **G** move cursor to end of the document
	- **w** go to next word 
	- **b** go to previous word
	- **^** move to star of current line
	- **$** move to end of current line
	- **dw** delete the current word
	- **O** open line above the cursor position
	- **o** open line below the cursor position
	- **:se number** show line numbers
	- **r** replacing the current character
- **vimtutor** provides an easy-to-use on-system tuto that guides you through the essentials of using **vi** and **vim**
