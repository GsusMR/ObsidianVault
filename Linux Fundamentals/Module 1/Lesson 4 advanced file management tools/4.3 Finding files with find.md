- **find / -name "hosts"** is searching for files with the exact name hosts
- **find / -name "hosts*"** search for files with the pattern hosts inside its name
- **find / -user "linda"** search for the files owned by the user linda 
- **find / -size +2G** search for files bigger than 2 gigabytes
- **find / -user "linda" -exec cp {} /root/linda \\;** copies all the results of the search to the indicated directory
- **find / -perm/4000** allows you to find all commands that have this specific permission set 

**_All the find options need to be used with permission (sudo before each sentence)_**
