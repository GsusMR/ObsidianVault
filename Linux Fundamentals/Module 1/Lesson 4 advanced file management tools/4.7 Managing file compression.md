- **gzip** is the most common compression utility. If you want to use gzip in tar, you use the **z** option. It gives a nice balance between compression ratio and tima that it takes to create the archive 
- **bzip2** or **-j** if you use it in tar. It offers a better compression, but it takes longer 
- **xzip** or **-J** is relatively new that offers the best possible compression but it is significantly slower
- **zip** can be used as well, and has a Windows-compatible syntax
