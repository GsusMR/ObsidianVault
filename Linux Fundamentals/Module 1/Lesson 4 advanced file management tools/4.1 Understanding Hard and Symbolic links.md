- A link is a file system entry that refers to another file or directory
- Hard links are pointing to the same inode on the same file system. An inode is what keeps all the administration of files, these are a part of the administration. An inode contains all the administration of the file, and these make a reference from the blocks. A block is a physical allocation unit on disc that a file is using
- A hard link is a name that directly refers to the inode, you put ir in the directory and it allows you to access the actual inode
- Symbolic links are shortcuts and add additional flexibility
	- They can exist on a directory
	- Cross-device symbolic links can be used
